Interface Matlab- PTV VISSIM 6.0

Interfaz para el control de PTV VISSIM 6.0 por COM desde Matlab

download .ZIPdownload .TGZ

Bienvenido al proyecto Interface Maltab-PTV VISSIM 6.0 basada en COM "KASHI".

Con el animo de poder explotar el poder de computo que ofrece Matlab, y el poder de simulación multinodal de PTV VISSIM se propuso diseñar una herramienta que permita utilizar el modelo de simulación de PTV VISSIM y que esta información pueda ser modificada y consultada por un script de Matlab. Trabajos similares se han realizado con otros lenguajes de programación (C++, Pyhton, C#,Java) y aun con Matlab,pero utilizando el esquema de objetos de la version 5.4 e inferiores. En vista de que no se encontro una forma de consulta de las herramientas de PTV VISSIM/Matlab o el esquema de clases COM de PTV VISSIM 6.0, se creo esta guia de consulta sobre el proyeto.
